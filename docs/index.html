What is this?

A REST API that provides Korean cosmetic ingredient data with regulatory compliance information across 10 countries. Sourced from MFDS (Korea's Ministry of Food and Drug Safety) and KCIA (Korea Cosmetic Industry Association). All data translated to English.

If you're formulating cosmetics for global markets or doing regulatory compliance work, this saves you from digging through government sites in languages you don't read.

API Endpoints

EndpointDescription
GET /v1/statsDatabase statistics
GET /v1/ingredient/inci?q=retinolSearch by INCI name
GET /v1/ingredient/cas?q=69-72-7Search by CAS number
GET /v1/ingredient/{code}/regulationsCountry-specific regulations
GET /v1/ingredient/search?q=glycerinPartial text search
