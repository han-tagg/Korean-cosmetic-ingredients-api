Frequently Asked Questions

Common questions about Korean cosmetic ingredient regulations and the K-Beauty API.

What cosmetic ingredients are banned in Korea?

Korea's MFDS (Ministry of Food and Drug Safety) prohibits 172+ cosmetic ingredients. These include substances classified as dangerous or insufficiently tested for cosmetic use. The full list can be queried via the API using /v1/ingredient/status?s=Prohibited.

How do I check if an ingredient is allowed in Korean cosmetics?

Search by INCI name, CAS number, or Korean name using the API. Each ingredient returns a regulation_status field showing whether it is Prohibited, Restricted, or Not Listed. For restricted ingredients, concentration limits and product-type conditions are also provided.

What is the difference between MFDS and KCIA?

MFDS is the Korean government regulatory agency that sets and enforces cosmetic ingredient regulations. KCIA (Korea Cosmetic Industry Association) maintains the cosmetic ingredient dictionary — the master list of 21,796 ingredients with INCI names, CAS numbers, and safety classifications. The K-Beauty API combines data from both sources.

How do Korean cosmetic regulations compare to EU?

Korea and the EU share a similar regulatory framework but differ in specific ingredient decisions. The EU bans 1,700+ substances compared to Korea's 172+. However, Korea has stricter age restrictions for some ingredients — Salicylic Acid is prohibited for children under 13 in Korea versus under 3 in the EU. Some ingredients like Isobutylparaben are banned in the EU but allowed in Korea. See our detailed comparison.

What is an INCI name and how do I search by it?

INCI (International Nomenclature of Cosmetic Ingredients) is the standardized naming system used worldwide. For example, "Retinol" is the INCI name for Vitamin A. Search the API using GET /v1/ingredient/inci?q=retinol.

Are Korean cosmetic regulations stricter than EU?

It depends on the ingredient. Korea is stricter for some — Salicylic Acid has an under-13 age restriction (EU: under 3). Korea is more permissive for others — Isobutylparaben is allowed at 0.4% (EU: banned entirely). Neither system is universally stricter. This is why multi-country compliance checking matters.

How often do Korean cosmetic regulations change?

MFDS typically updates its cosmetic ingredient regulations several times per year. The API monitors source data weekly using automated compliance checking, and the database is updated whenever MFDS or EU CosIng publishes changes.

What concentration limits apply to retinol in Korea?

Retinol (CAS: 11103-57-4) is classified as Restricted in Korea and regulated as a functional cosmetic active ingredient. Specific concentration limits depend on the product type. Query the API for full regulatory details.

How many countries does the API cover?

The API covers 10 countries and regions: Korea, EU, China, Japan, ASEAN, Canada, Brazil, Argentina, Taiwan, and USA. The database contains 30,960 regulation records, allowing you to compare how the same ingredient is regulated across markets with a single call. See the country-by-country breakdown.
